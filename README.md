D4SG-Frontend

# Install
```
npm install
```

# First Init

```
gulp build
```

Then Copy `docs/bower_components` folder to `dist/bower_components`.
Then Copy `docs/imgs` folder to `dist/imgs`.
Then Copy `docs/*.pdf` folder to `dist/*.pdf`.

The setup has already.

gulp
